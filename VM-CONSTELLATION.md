# Horror$Place VM-Constellation: Repository Tags and Short Descriptions

## HorrorPlace-Codebase-of-Death  `[Tier-2 · Agents]`

Private: internal, non‑GitHub distribution.

Tag: `AGENTS_CORE`

Short description:  
Houses full AI personalities and underground agent implementations (Archivist/Witness/Echo/Threshold/Process variants) used to drive high‑SHCI spectral behavior, while remaining bound to public schemas and invariant APIs defined in `Horror.Place`.[file:5][file:6]
