## How engines and Lua should use these presets

Each preset becomes a tag in the audio policy layer, not a direct file reference. [ppl-ai-file-upload.s3.amazonaws](https://ppl-ai-file-upload.s3.amazonaws.com/web/direct-files/collection_cdb90fc3-8a6a-46e2-89a6-187b2f85f988/d0db60df-4dcc-43b2-b889-2eb4a88581ea/this-research-focuses-on-creat-hfrKYptnQR.7lVWEu2zdUg.md)

In `enginelibrary/horroraudio.lua`, you would:

- Compute local invariants via `H.sampleAll(regionId, tileId, playerId)`. [ppl-ai-file-upload.s3.amazonaws](https://ppl-ai-file-upload.s3.amazonaws.com/web/direct-files/collection_cdb90fc3-8a6a-46e2-89a6-187b2f85f988/d0db60df-4dcc-43b2-b889-2eb4a88581ea/this-research-focuses-on-creat-hfrKYptnQR.7lVWEu2zdUg.md)
- Select a preset or blend based on ranges above (e.g., high CIC + high AOS + high LSG → `Black_Hallways`). [ppl-ai-file-upload.s3.amazonaws](https://ppl-ai-file-upload.s3.amazonaws.com/web/direct-files/collection_cdb90fc3-8a6a-46e2-89a6-187b2f85f988/d0db60df-4dcc-43b2-b889-2eb4a88581ea/this-research-focuses-on-creat-hfrKYptnQR.7lVWEu2zdUg.md)
- Return an abstract sequence such as `{ preset = "A12.Black_Hallways", weight = 0.8, layers = { "low_drones", "door_slams_sparse" } }` for the C adapter to translate into RTPCs and concrete audio.
