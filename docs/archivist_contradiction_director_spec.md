## 2. Data Model: Storylets and Invariant Binding

### 2.1 History as Schema-Constrained Storylets

Every region’s history is represented as **schema-constrained storylets**: [ppl-ai-file-upload.s3.amazonaws](https://ppl-ai-file-upload.s3.amazonaws.com/web/direct-files/collection_cdb90fc3-8a6a-46e2-89a6-187b2f85f988/114f9185-3edf-4251-b08f-db275e57eb2f/for-this-conversation-let-s-cr-84g4wybTQday6rTQXVEVCQ.md)

- `time`: when the event occurred (absolute/relative).  
- `cause`: what triggered it (industrial accident, ritual, disaster).  
- `effect`: consequence (disappearances, contamination, cover-up).  
- `invariants`: links to CIC, MDI, AOS, RRM, FCF, SPR, RWF, SHCI.  
- `tags`: mythic motifs, institutional actors, emotional tone.

These storylets are **pre-validated**:

- They respect local invariant ranges (no event can be attached to a region that contradicts its CIC/MDI/AOS profile).  
- They match **region archetype templates** (industrial valley, irradiated marsh, sealed town, etc.). [ppl-ai-file-upload.s3.amazonaws](https://ppl-ai-file-upload.s3.amazonaws.com/web/direct-files/collection_cdb90fc3-8a6a-46e2-89a6-187b2f85f988/114f9185-3edf-4251-b08f-db275e57eb2f/for-this-conversation-let-s-cr-84g4wybTQday6rTQXVEVCQ.md)

The Archivist **cannot invent arbitrary spookiness**; it can only:

- Select, reorder, redact, and cross-reference **existing, validated events**.  
- Inject bounded contradictions by juxtaposing conflicting but individually plausible records.

For the Archivist, this loop becomes **informational**:

- **Hesitation near high-AOS zones** (slow approach, frequent camera sweeps) → proxy for rising UEC.  
- **Repeated rereads of conflicting logs** → strong signal for EMD and CDL rising.  
- **Backtracking for “one more file”** or archive access → indicates EMD is high but not yet overwhelming.

The Archivist monitors these signals per region, adapting:

- **Redaction ratio:** how many lines or fields are hidden.  
- **Contradiction rate:** how often multiple explanations appear.  
- **Clarifying hint density:** how many small, convergent clues it allows.

## 6. Lua-Level Integration Pattern (High-Level Sketch)

The MASTERBLUEPRINT already defines a narrow Lua API:

- `H.*` for invariants.  
- `BCI.*` for affect.  
- `Director.*` for pacing and Surprise.Events!. [ppl-ai-file-upload.s3.amazonaws](https://ppl-ai-file-upload.s3.amazonaws.com/web/direct-files/collection_cdb90fc3-8a6a-46e2-89a6-187b2f85f988/114f9185-3edf-4251-b08f-db275e57eb2f/for-this-conversation-let-s-cr-84g4wybTQday6rTQXVEVCQ.md)

The Archivist is an `A.*` module that respects strict boundaries:

- It **never** accesses raw region tables directly.  
- It uses:
  - `H.*` for invariant queries.  
  - `Policy.*` for entitlement and Rivers of Blood constraints.  
  - `Metrics.*` for UEC/EMD/STCI/CDL/ARR estimates.  
  - `Story.*` for schema-validated storylets.

A minimal integration sketch:

```lua
-- File: engine/archivist.lua

local H       = require("engine.horror_invariants")
local Metrics = require("engine.metrics")
local Story   = require("engine.story_spine")
local Policy  = require("engine.entitlement_policy")

local A = {}

function A.update(region_id, dt)
    if not Policy.is_archivist_enabled(region_id) then
        return
    end

    local inv = {
        cic  = H.CIC(region_id),
        aos  = H.AOS(region_id),
        rwf  = H.RWF(region_id),
        spr  = H.SPR(region_id),
        shci = H.SHCI(region_id)
    }

    local m = {
        uec = Metrics.UEC(region_id),
        emd = Metrics.EMD(region_id),
        cdl = Metrics.CDL(region_id),
        arr = Metrics.ARR(region_id)
    }

    local mode = A.select_mode(inv, m)         -- e.g., reveal, redact, contradict
    local beat = Story.next_unseen_beat(region_id, { mode = mode })

    if beat then
        local fragment = A.transform_beat(beat, inv, m, mode)
        Story.surface_fragment(region_id, fragment)
    end
end

return A
```

- `select_mode` uses AOS/RWF and UEC/ARR to decide whether to **clarify**, **withhold**, or **contradict**.  
- `transform_beat` applies redaction and contradiction templates tied to policy and entitlement contracts.  
- All surfaced fragments are **auditable** against invariant and policy constraints.

Narratively, this makes the Archivist:

- Less a “character” and more a **mechanism**—a god-like indexing process deciding which history you are allowed to see, and in what warped order, under a measurable, ethically bounded contract.
