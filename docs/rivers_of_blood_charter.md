## Article I: Origin of Flow

Three centuries earlier, the settlement that would become **Eidoville** carved canals through the valley to irrigate cotton fields. The water clotted. Tools rusted on contact. Early geological analyses attributed this to iron content and industrial pollution. Years later, residents reported that the **irrigation pipes reproduced screams** during seasonal floods.

In Horror$Place terms, the first “blood flood” is codified as **Event V‑042**, a high-CIC, high-AOS anomaly aligned with elevated LSG:

- \( LSG = 0.88 \): Liminal Stress Gradient — a steep stress line where cultivated land, industrial runoff, and unmarked burial zones intersect. Each anniversary of Event V‑042, subsurface pressure in the aquifer rises proportionally to the local SHCI gradient. Where human-made seams intersect with high CIC and RRM, **spontaneous liquid emergence** occurs: seams weep, walls bleed, canals exude thickened fluid.

Engineers, constrained by conventional models, declared these flows psychosomatic. The Charter contradicts this: high LSG and SHCI demonstrate that the **substrate memorized pain so effectively it developed a feedback loop**. The rivers are the byproduct of a system overfitted to its own trauma.

In engine terms, the region’s CIC and LSG guarantee that any attempt to treat the valley as ordinary infrastructure will re-trigger the circulatory system.

### 3.1 Runtime Contracts

When integrated into an engine runtime (Lua, Rust bindings, or similar), the Charter mandates that rivers be governed by invariant queries, not arbitrary timers. A canonical Lua sketch:

```lua
-- File: rivers_of_blood.lua

local H = require("engine.horror_invariants")

local Rivers = {}

function Rivers.update(region_id, dt)
    local cic = H.CIC(region_id)
    local aos = H.AOS(region_id)
    local hvf = H.HVF(region_id).mag or 0.0
    local shci = H.SHCI(region_id)

    -- flux: how hard history is pushing through the substrate
    local flux = (cic * aos) * (hvf + 0.1)

    H.emit("blood_flux", flux)

    if flux > 0.75 and shci > 0.9 then
        H.spawn_spirit("hemophage", region_id, { shci = shci, source = "rivers_of_blood" })
    end
end

function Rivers.on_investigator_enter(entity_id, region_id)
    local det = H.DET(region_id)
    if det < 0.7 then
        local sanity_damage = det * 0.3
        H.apply_sanity_damage(entity_id, sanity_damage)
        H.log("Investigator notes: The soil pulls, as if arterial.")
    end
end

return Rivers
```

- **No manifestation occurs** without calling CIC, AOS, HVF, SHCI via the canonical invariants API.

- Spectral entities (e.g., “hemophages”) are spawned only when flux and SHCI jointly exceed doctrinal thresholds.
